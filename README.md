## Run Locally

**Prerequisites:**  [Android Studio](https://developer.android.com/studio)


1. Open Android Studio
2. Select **Open** and choose the directory containing this project
3. Allow Android Studio to fix any incompatibilities as it imports the project.
4. Create a file named `.env` in the project directory and set `GEMINI_API_KEY` in that file to your Gemini API key (see `.env.example` for an example)
5. Remove this line from the app's `build.gradle.kts` file: `signingConfig = signingConfigs.getByName("debugConfig")`
6. Run the app on an emulator or physical device

# 📖 About

**Merge6x6** is a next-generation number-merging puzzle game inspired by classic merge mechanics, redesigned with a modern cyber aesthetic and buttery-smooth gameplay.

Every move requires strategic thinking as you place numbered blocks, trigger satisfying chain reactions, unlock achievements, and climb toward the legendary **2048 tile—and beyond**.

The project is built as a **fully self-contained browser game** using **HTML5, CSS3, and Vanilla JavaScript**, while also running as a **native Android application** through **Jetpack Compose + Hardware Accelerated WebView**.

---

# 🎮 Gameplay

* Place numbered blocks onto a **6×6 board**
* Merge matching adjacent blocks
* Create explosive chain reactions
* Earn combo multipliers
* Reach higher powers of two
* Beat your personal best
* Compete in Daily Challenges

Simple to learn.

Difficult to master.
