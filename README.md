# Mega 6x6

Mega 6x6 is a modern browser-based number-merging puzzle game inspired by classic merge mechanics.

## Features

- 🎮 6×6 strategic gameplay
- ✨ Smooth merge animations
- 🏆 High score system
- 💾 Auto Save
- ↩ Undo Move
- 🎯 Combo Multipliers
- 📱 Mobile Friendly
- 🌙 Dark Theme
- ⚡ Fast Performance
- 🎨 Original UI Design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Future Plans

- Online Leaderboards
- Daily Challenges
- New Game Modes
- Achievements
- Cloud Save
